# Cockpit 와이어프레임 (텍스트)

UI 프레임워크와 무관한 **구조 초안**. 구현 시 이 배치를 따른다.

---

## S01 Overview (홈)

```text
┌─────────────────────────────────────────────────────────────┐
│  TradingBot  cockpit                    [긴급정지: ON ●]   │
├─────────────────────────────────────────────────────────────┤
│  ┌──────────────────────┐  ┌─────────────────────────────┐ │
│  │ 실거래 잠금          │  │ 봇 상태                     │ │
│  │ 🔒 LIVE LOCKED       │  │ 읽기 준비 / 신호 대기 등    │ │
│  │ 실제 주문 불가       │  │ 한 문장 설명                │ │
│  └──────────────────────┘  └─────────────────────────────┘ │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 지금 내가 할 일                                      │   │
│  │ · (예) 토스 연결 전 — Phase 2 진행 대기              │   │
│  │ · 막을 일 없음 / 또는 경고 확인                      │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  연결: 미연결/읽기전용/오류    시장: 장중/장외/모름         │
│  계좌: ****1234  평가: ---     후보: 0건 (실주문 아님)     │
│                                                             │
│  최근 차단·경고                                             │
│  · kill switch 활성 — 실거래 차단                           │
│  · (데이터 오면) 시세 지연 등                               │
│                                                             │
│  최근 기록 (3)                                              │
│  · 09:00 시스템 시작 — dry-run 모드                         │
│  · ...                                                      │
│                                                             │
│  [상태 자세히] [주문 후보] [기록] [설정]                    │
│  ※ “실주문 실행” 버튼 없음                                  │
└─────────────────────────────────────────────────────────────┘
```

## S03 Live Lock + Kill Switch

```text
┌──────────────────────────────────────────┐
│ 돈 보호 장치                              │
│                                          │
│ Kill Switch:  [ ON  ]  ← 켜기는 쉬움      │
│ 설명: 긴급 정지. ON이면 실거래 불가       │
│                                          │
│ Live Lock:    [ 잠김 ]                   │
│ 설명: 실주문 경로 잠금                    │
│ 잠금 해제: Phase 6 전 비활성              │
│                                          │
│ 현재 주문 모드: dry_run                  │
│ ALLOW_LIVE_ORDERS: false                 │
└──────────────────────────────────────────┘
```
