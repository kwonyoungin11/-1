# 자동매매 콕핏 — Mac 데스크톱 앱

## 화면 배치
- **위 2/3**: 차트 (트레이딩뷰 스타일 캔들 + 매수/매도 표시)
- **아래 1/3**: 자동매매 조작 UI만
  - 대상 주식 종류
  - 매매 전략
  - 자동매매 시작 / 종료
  - 잔액
  - 수익률

## 실행
```bash
cd "/Users/kwon/Documents/c#"
dotnet run --project src/TradingBot.App
```

## 안전
- 시작/종료 = **연습 세션** 제어 (실주문 차단 유지)
- 차트 매수/매도 = **표시 마커** (주문 실행 버튼 아님)
- 잔액·수익률 = **연습 기준** 표시
