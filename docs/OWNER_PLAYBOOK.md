# 오너 플레이북 (비개발자용)

## 이 프로그램이 하는 일 (한 줄)

토스증권 Open API로 **나스닥 시세·계좌를 안전하게 읽고**, 전략 신호와 주문 **후보**를 만들되, **실제 주문은 기본 차단**합니다.

## 네 가지 질문으로 모든 보고 읽기

1. 지금 무엇을 하려는가?
2. 왜 필요한가?
3. 돈을 잃을 위험이 있는가?
4. 내가 결정해야 할 것은 무엇인가?

## 안전 기본값 (변경 금지 권장)

| 설정 | 기본값 | 의미 |
|------|--------|------|
| ALLOW_LIVE_ORDERS | false | 실주문 허용 안 함 |
| KILL_SWITCH | true | 긴급 정지 ON |
| ORDER_MODE | dry_run | 연습(모의)만 |

## 오늘 할 수 있는 일 / 하면 안 되는 일

**할 수 있음**

- `.env`에 토스 `client_id` / `client_secret` 직접 입력 (에이전트에게 값 알려주지 말 것)
- Mac에 .NET SDK 설치
- 문서 읽기, cockpit 화면 방향 결정

**하면 안 됨**

- 실주문 버튼 요청
- kill switch 끄기 요청 (아직)
- API 키를 채팅에 붙여넣기
- “수익 보장” 전략 기대

## 비밀키 규칙

1. 키는 오직 본인 Mac의 `.env`에만 입력
2. 채팅/메일/스크린샷에 키 금지
3. `.env`는 git에 올라가면 안 됨 (자동 검사함)

## 사고가 나면

`docs/INCIDENT_RESPONSE.md` 를 따르세요. 요약:

1. 프로그램 중지
2. kill switch 유지/확인
3. 네트워크 끊기 (필요 시)
4. 로그 수집 (비밀 마스킹)
5. 원인 기록

## 기술 용어 카드 예시

**기술 용어:** fail-closed  
**쉬운 설명:** 잘 모르겠으면 주문을 막고 봄  
**중요한 이유:** 오류 시 “일단 주문”이 가장 위험  
**잘못되면:** 예상 밖 매수/매도  
**현재 안전장치:** 기본 차단, kill switch, live gate 테스트
