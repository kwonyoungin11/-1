**투자 조언이 아닙니다.**  
**과거 시뮬레이션 ≠ 미래 수익.**  
**실주문은 기본 차단입니다.** (후보·연습만)

대상: 비개발자 오너 · 자동매매 콕핏 화면을 보며 상태를 이해할 때

## 1. 한 줄로

**CERS(비용 인식 평균회귀)** 는  
“지금 가격이 단기 평균보다 **충분히** 아래로 빠졌고,  
그 되돌림 여지가 **왕복 거래비용(약 0.3%)의 2배 이상**으로 보일 때만”  
**롱(매수) 주문 후보**를 만듭니다.

- 숏 없음  
- 후보 ≠ 체결 ≠ 실주문  
- 잘 모르겠거나 데이터가 없으면 **안 함(차단)**

| 상황 | 오너가 할 일 |

| 실주문 하고 싶을 때 | live readiness · 게이트 · 본인 승인 전부 전부 통과 전 **불가능** |

## 3. 화면에 보이는 숫자 (기억할 것)

| 표시 | 의미 | 오너 해석 |
|------|------|-----------|
| **expected** | 추정 “되돌림 여지”(비율) | 높을수록 가설상 여지 큼 · **보장 아님** |
| **thr / 문턱** | **0.0060** (0.60%) | expected가 이걸 **넘을 때만** 진입 후보 |
| **왕복 비용** | 약 **0.30%** | 수수료+슬리피지 가정 (토스 미국주식 근사) |
| **상태: 관망** | 조건 미달·쿨다운·데이터 부족 | 주문 없음 → 정상 |
| **상태: 진입가능** | expected > 문턱 | 후보 생성 가능 (연습 모드에서 기록) |
| **상태: 보유** | 이미 롱 포지션 가정 | 청산 규칙만 봄 |
| **손절 SL** | 진입가 × **0.988** (−1.2%) | 더 빠지면 나가려는 선 |
| **익절 TP** | 진입 시 expected × 1.5 만큼 | 진입 때 여지에 비례 (고정 % 아님) |
| **보유 봉** | 최대 **40** × 1분 | 약 40분 안에 안 되면 시간 청산 가설 |
| **EMA21** | 21분 평균선 | 가격이 다시 이 선 위로 오면 청산 가설 |

상태 줄 예시:

```text
CERS expected=0.0072 thr=0.0060 · 진입가능
CERS expected=0.0041 thr=0.0060 · 관망
CERS 보유 12/40봉 · SL … · TP …
```

## 4. 규칙 요약 (진입 / 청산)

### 들어가려 할 때 (롱 후보)

1. 1분봉 데이터가 **충분·신선**할 것  
2. 가격이 EMA21 **아래** (평균보다 싼 쪽)  
3. **expected > 0.0060**  
4. 직전 청산 후 **3분(3봉) 쿨다운**이 아닐 것  
5. 세션·킬스위치·일손실·리스크 게이트가 **통과**일 것  

→ 통과 시에만 **지정가 매수 후보** (+ 손절·익절 브래킷)

### 나가려 할 때 (보유 중)

아래 **하나라도** 해당하면 매도 후보 가설:

| 조건 | 쉬운 말 |
|------|---------|
| −1.2% | 손절 |
| + (진입 expected × 1.5) | 익절 (진입 때 본 여지의 1.5배) |
| 종가 ≥ EMA21 | 평균까지 되돌아옴 |
| 보유 ≥ 40봉 | 시간 초과 |

## 5. 연구 성적 (정직하게)

대상 리포트: `artifacts/vmar_cers_6m_backtest_report.md`  
(VMAR · 1분 · 약 6개월 · **과거 시뮬** · 실주문 경로 아님)

| 항목 | 값 |
|------|-----|
| 전략 | CERS (연구 1위) |
| 총수익(시뮬) | **+738.1%** |
| 최대낙폭 MDD | **51.7%** |
| 거래 수 | 556 |
| 승률 | 약 60.8% |
| 같은 기간 보유만(BuyHold) | 약 **−98.5%** (급락 구간) |

**꼭 읽을 주의:**

1. 과거 숫자일 뿐, **내일 수익 보장 아님**  
2. 시뮬 체결 = 다음 봉 시가 가정 → 실제 호가·미체결과 **다름**  
3. MDD 50%대 = 중간에도 **크게 빠질 수 있음**  
4. 저유동 종목은 가정 슬리피지보다 **훨씬 나쁜 체결** 가능  
5. 백테스트 1위여도 **실주문 잠금은 그대로**
