# Copywriting Guide

## Voice

- Calm, precise, non-hype
- Korean primary for owner UI
- No profit promises

## Examples

| Bad | Good |
|-----|------|
| 지금 매수하세요 | 주문 후보가 생성되었습니다 (실주문 아님) |
| 수익 시작 | paper trading 기록 중 |
| 실거래 ON | live blocked — 승인 필요 |

## Error copy

Always include: what happened, impact on money (usually none if blocked), what to do next.
