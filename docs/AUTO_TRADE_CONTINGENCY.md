# 자동매매 컨틴전시 · 미체결 · 뉴스

**투자 조언 아님 · 실주문 잠금.**  
전략 기준: `SpacexOfficialStrategyPreset`.

## 지정가 미체결

| 상황 | 대응 |
|------|------|
| TTL 만료 (기본 2×15m) | 취소 후 재평가 |
| 시장이 지정가에 유리 | 유지 또는 ATR×0.25 이내 재호가 |
| 시장이 불리 이탈 | 취소 · 시장가 추격 금지 |
| 부분 체결 | 잔량 취소 또는 SL/TP 재계산 |
| 신호 반전 / 킬스위치 / 데이터 오류 | 미체결 취소 · 신규 금지 |

코드: `LimitOrderLifecyclePolicy`, `WorkingOrderDecision`.

## 변수 대응

변동성·횡보·갭·거래량·세션·일손실·잔고·API오류·경고·뉴스데이 → **진입/유지/취소/감축/정지** 중 하나로 고정. 애매하면 관망.

## 필수 vs 비필수

- **필수:** 시세, 규칙, 리스크, 주문 수명, 체결 후 관리, 킬스위치, 감사, 수수료 인지  
- **비필수:** 뉴스 감성 매수, 지표 과다, 시장가 기본, 장식 UI  

## 뉴스·공시

- 토스 OpenAPI: 헤드라인 뉴스 피드 없음 → `warnings` + **오너 뉴스데이 토글**  
- 사용법: **차단·사이즈 축소·쿨다운** (방향 자동 결정 금지)  
- 외부 뉴스: 읽기 전용 패널 후순위
