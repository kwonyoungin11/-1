# ADR 0001: Grok 4.5 C# trading harness first

## Status
Accepted — 2026-07-09

## Context
Greenfield repo. Owner is non-developer. Goal is safe Toss NASDAQ automation.

## Decision
Build harness (docs, safety defaults, scans, solution skeleton) before feature trading.

## Consequences
Slower first “feature”, much lower accidental-order risk.
